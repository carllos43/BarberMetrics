Atualizar o aplicativo BarberMetrics para funcionar como um sistema profissional de controle de produtividade e faturamento para barbeiros.

OBJETIVO
Criar um aplicativo simples, rápido e confiável para registrar atendimentos, acompanhar produtividade, analisar faturamento e controlar despesas da barbearia.

────────────────
1. SISTEMA DE TEMPO PROFISSIONAL

Substituir qualquer cronômetro simples por um sistema baseado em timestamp.

Ao iniciar corte:

const startTime = Date.now()

Ao finalizar:

const endTime = Date.now()

Calcular duração:

duration = endTime - startTime

Salvar no banco:

start_time_utc
end_time_utc
duration_ms

Sempre salvar timestamps em UTC.

────────────────
2. CORREÇÃO DE FUSO HORÁRIO

Exibir todos os horários no horário do Brasil.

Usar:

new Date(timestamp).toLocaleTimeString("pt-BR", {
 timeZone: "America/Sao_Paulo",
 hour: "2-digit",
 minute: "2-digit"
})

Exemplo no histórico:

21:16 → 21:33
17 min

Salvar startTime no localStorage.

localStorage.setItem("activeCutStart", startTime)

const savedStart = localStorage.getItem("activeCutStart")

Recalcular o tempo automaticamente.

Garantir funcionamento mesmo se:

- celular bloquear

- atualizar página

11. CÁLCULOS

Tempo atendendo:
soma das durações.

Tempo trabalhando:

Tempo parado:
tempo_trabalhando - tempo_atendendo.

Produtividade:

produtividade = tempo_atendendo / tempo_trabalhando
